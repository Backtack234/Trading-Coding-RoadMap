# Trading Roadmap PWA

Your 6-month Trading + Python study plan as an iPhone app.

## How to Install on Your iPhone (5 minutes)

### Step 1: Create a GitHub account (if you don't have one)
1. Go to github.com on your phone
2. Sign up for free

### Step 2: Create a new repository
1. Tap the **+** button → **New repository**
2. Name it: `trading-roadmap`
3. Make sure **Public** is selected
4. Tap **Create repository**

### Step 3: Upload the files
1. In your new repo, tap **"uploading an existing file"** link
2. Upload all 3 files from this folder:
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. Tap **Commit changes**

### Step 4: Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under "Source", select **main** branch
3. Folder: **/ (root)**
4. Tap **Save**
5. Wait 1-2 minutes. Your site will be at:
   `https://YOUR-USERNAME.github.io/trading-roadmap/`

### Step 5: Add to iPhone Home Screen
1. Open the link above in **Safari** (must be Safari, not Chrome)
2. Tap the **Share button** (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Name it whatever you want and tap **Add**

Done! You now have an app icon on your home screen that works offline.

## Features
- Mark weeks as complete (saved automatically)
- Add notes for each week
- Works offline after first load
- Progress tracking across all 24 weeks
- Your schedule mapped to every time slot
